# 类型基础

## 编写你的第一个TypeScript程序

1. 初始化 `npm` 项目

    ```js
      npm init -y
    ```

2. 全局安装 `typescript`

    ```js
    npm install typescript -g
    ```

3. 在当前目录初始化 `typescript` 项目

    ```js
    tsc --init
    ```

4. 尝试看看能不能编译 `ts` 文件

    ```js
    touch index.ts
    tsc index.ts
    ```

5. 搭建 `typescript` 工程化项目

    ```js
    npm install webpack webpack-cli webpack-dev-server -D
    ```
